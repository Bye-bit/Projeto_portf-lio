# 🚀 Portfólio - Lucas Fernandes

Este é o meu site de portfólio desenvolvido com **Next.js**, com o objetivo de apresentar meus projetos, informações pessoais e demonstrar minhas habilidades em desenvolvimento web.

---


##  Sobre o projeto

O site funciona como um currículo online, contendo:

*  Seção "Sobre mim"
*  Lista de projetos
*  Links para redes sociais
*  Página extra com um **Jogo da Forca interativo**

---

## Jogo da Forca

O projeto inclui uma implementação completa do jogo da forca, com:

* Exibição da palavra com letras ocultas
* Entrada de letras via teclado virtual
* Controle de tentativas
* Lista de letras já utilizadas
* Mensagem de vitória ou derrota
* Botão para reiniciar o jogo
* Palavras aleatórias (lista com +30 palavras)

##  Como rodar o projeto

```bash
# Clonar o repositório
git clone (https://github.com/Bye-bit/Projeto_portf-lio.git)

# Entrar na pasta
cd portifolio

# Instalar dependências
npm install

# Rodar o projeto
npm run dev
```
